About this prototype. Pick a whole groupset / wheelset / brake set / cockpit kit (green dashed cards) to drop in all the parts at once — that group is then priced as the bundle; swap any single part and it reverts to per-part pricing. Every card shows a compatibility dot (green = no conflict found, yellow = check the warning first, red won’t fit, grey = nothing selected yet) and a sample weight, and the catalog is sorted compatible-first (use the Sort menu to reorder by name, price or weight). Specs, prices and weights are illustrative samples unless a part shows a green ✓ Verified badge (checked against the manufacturer, with a source link). Logic lives in src/compat.js; run node validate.js to check the data and npm test for the suite. Mullet (29 front / 27.5 rear) builds work on mullet-capable frames — try the Mullet build demo or pick a 27.5 rear wheel + tire on the YT Capra.

⚐ Report a wrong verdict

Spotted a fit we got wrong, or a wrong spec? Real builds are how these rules get checked. This sends a description of your current build — no account needed.

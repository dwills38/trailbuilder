# Manufacturer master list — the breadth-chip reference

**Source: Douglas, 2026-07-24.** His own list of manufacturers to work from when writing
BREADTH chips. Read this before scoping any breadth or cross-category wave.

## How to use it

- This is a **candidate pool, not a to-do list.** A name here is a brand worth *checking*,
  never evidence that the brand belongs in a given catalog or category.
- **Always re-derive the actual gap from the live catalog file** before writing a chip.
  Chip scopes go stale within hours — same-day waves have already covered brands that a
  day-old audit still lists as missing.
- A brand that turns out **not to make the part** is a FINDING, not a gap. Record it in the
  worker report and move on. Padding a category to hit a number is the failure this list
  must not cause. (Examples already established: Chris King makes no rims; Nukeproof sells
  no standalone hub shell; PNW makes no saddle or rigid post; Hunt makes only wheels;
  microSHIFT makes no chains; Bell is helmets-only; Smith is eyewear + helmets only.)
- **Absence from this list is not evidence a brand is out of scope** — it is Douglas's
  working list, not an exhaustive registry.
- Catalog residency still follows hard rule 1: a part lives in the catalog for its own type.
  The only sanctioned dual-catalog case is a genuine road/gravel crossover bike, which may
  carry its own row in each with its own id.
- Some sections below are **categories the catalogs do not currently model** (computers /
  power meters / electronics, tools / maintenance, e-bike motor systems as standalone
  products). Do NOT invent a category or widen a schema to fit them — that is a separate
  reviewed decision for Douglas.
